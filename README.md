##  Fitur Utama
Aplikasi ini mencakup berbagai modul utilitas sistem untuk berbagai tipe perangkat:
* **Auto-Detection**: Sistem pemindaian otomatis untuk mendeteksi perangkat dalam mode aktif maupun protokol komunikasi sistem.
* **System Navigation**: Navigasi cepat untuk berpindah antar mode seperti Bootloader, Recovery, dan sistem utama.
* **Flash Engine**: Antarmuka khusus untuk pemasangan file citra sistem pada berbagai partisi (Boot, Recovery, VBMeta, System, dll.).
* **Maintenance Tools**: Modul untuk pembersihan data sistem dan pemulihan ke pengaturan pabrik (Factory Reset).
* **Premium Interface**: Estetika modern dengan indikator proses neon dan log konsol real-time untuk pemantauan aktivitas.

##  Persiapan & Penggunaan
Untuk memastikan integritas fungsi aplikasi pada perangkat Anda, ikuti langkah-langkah berikut:
1. Unduh paket aplikasi terbaru dari bagian Releases.
2. Pastikan komponen binari sistem (seperti adb dan fastboot) tersedia di dalam folder yang sama dengan file aplikasi utama.
3. Jalankan aplikasi dan hubungkan perangkat Android Anda menggunakan kabel data berkualitas tinggi.
4. Gunakan tombol DETEKSI PERANGKAT untuk memulai sesi manajemen sistem.
